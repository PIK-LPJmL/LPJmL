     NAME
          regridlpj - regrid CRU input data set for selected countries

     SYNOPSIS
          regridlpj [-dir path] filesuffix outfile countrycode ...

     DESCRIPTION
          Program regrids CRU input files for selected countries and
          creates corresponding input configuration file.

     OPTIONS
          -dir path
               set directory for regridded input files

          filesuffix
               File suffix for regridded input files and configuration
               file

          countrycode
               ISO 3166-1 alpha-3 abbreviations for countries or
               rountry codes as numbers defined in include/managepar.h

     EXAMPLES
          Create input data files for USA:
               regridlpj usa USA

     EXIT STATUS
          Non zero is returned in case of failure.
